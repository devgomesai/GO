# Learning Go

This repository contains examples and exercises for learning the Go programming language. It covers fundamental Go concepts through practical examples organized by topic.

## Overview

This collection of Go programs demonstrates basic to intermediate concepts in Go programming, including:

- Basic syntax and data types
- Collections (arrays, slices, maps)
- String manipulation
- Structs and methods
- Interfaces
- Error handling

## Getting Started

To run any of the examples:

1. Make sure you have Go installed on your system
2. Navigate to the specific example directory
3. Run `go run main.go`

For example:
```bash
cd cmd/basics
go run main.go
```

## Examples

- `cmd/basics`: Covers basic Go syntax, variables, data types, and error handling
- `cmd/collections`: Demonstrates arrays, slices, and maps
- `cmd/strings`: String manipulation and processing techniques
- `cmd/structs`: Structs, methods, and interfaces in Go

## Prerequisites

- Go 1.19 or higher
